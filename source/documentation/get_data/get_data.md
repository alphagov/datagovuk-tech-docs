# Get data

You can use the data directory to [search and browse datasets by data owners across the public sector](https://data.gov.uk/search).

Alternatively, you can use an API to search and get data from the data directory.

[Curated data pages](https://data.gov.uk/collections) are not yet available via the API. [Check out the roadmap](https://data.gov.uk/roadmap) to learn more about upcoming changes to data.gov.uk.
