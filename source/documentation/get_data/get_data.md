# Get data

You can use the data.gov.uk directory to [search and browse datasets](https://data.gov.uk/search) published by data owners from across the public sector.

Alternatively, you can use an API to search and get data from the directory.

Data from the [curated collections](https://data.gov.uk) is not yet available via the API. [Check out the roadmap](https://data.gov.uk/roadmap) to learn more about upcoming changes to data.gov.uk.
