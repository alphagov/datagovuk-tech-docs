# data.gov.uk technical documentation

data.gov.uk helps people find and use open data from the public sector. The website began transformation in March 2026 with a new design and new features:

•	curated data collections of high quality, on-demand data
•	a data manual highlighting best practices for people who work with data in the public sector
•	a section titled the data directory 

The section titled data directory provides equal functionality to the data.gov.uk service before the March 2026 transformation. On it publishers across government make public sector open data available in a self-selecting manner through a CKAN catalogue.

We plan to bring the data directory closer to the new data collections as we progress the transformation. Find out more [in our roadmap](https://data.gov.uk/roadmap).

In the meantime, you can follow this guidance to: 

* [publish and manage datasets](publish_and_manage_data/) on the data.gov.uk directory
* [get data](get_data/) from the data.gov.uk directory using an API

If you can’t find what you’re looking for, [contact data.gov.uk support](https://data.gov.uk/support).
